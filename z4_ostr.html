1) Kiedy powstał JavaScript (podaj datę).
JavaScript został stworzony w 1995 roku przez Brendana Eicha.
2) Dlaczego jest to język skryptowy.
JavaScript jest językiem skryptowym, ponieważ jest interpretowany przez przeglądarki internetowe w czasie rzeczywistym.
3) Gdzie jest wykonywany JS (klient lub serwer).
JavaScript może być wykonywany zarówno po stronie klienta, jak i serwera (np. przy użyciu Node.js).
4) Dlaczego JS to bezpieczne narzędzie.
JavaScript działa w izolowanym środowisku przeglądarki, co chroni przed dostępem do systemu plików i zasobów użytkownika.
5) Czy jest to język obiektowy, jeśli tak to dlaczego.
Tak, JavaScript obsługuje obiekty, umożliwia ich tworzenie i manipulowanie nimi oraz wspiera dziedziczenie i polimorfizm.
6) Wymień zalety JS:
1. Łatwość uczenia się i używania.
2. Wsparcie dla programowania asynchronicznego.
3. Możliwość uruchamiania kodu po stronie klienta i serwera.
7) Framework to:
Zbiór narzędzi i bibliotek ułatwiających tworzenie aplikacji.
8) Do aplikacji mobilnych można użyć Framework.
Tak, np. React Native, Ionic lub Flutter.
9) Aplikacje responsywne to:
Aplikacje, które dostosowują się do różnych rozmiarów ekranów.
