1. W jaki sposób można określić właściwość danego obiektu?

Sposób 1: Użycie zapisu postaci:

        auto.marka = "Opel Omega";
        auto.rok = 1996;
        auto.cena = 25000;

Sposób 2: Użycie notacji tablicowej:

        auto["cena"] = 25000;

2. W jaki sposób można odwołać się do pól i metod?

        console.log(samochod1.kolor);

3. Wymień cechy charakterystyczne konstruktora

  • Jest wywoływany automatycznie przy tworzeniu obiektu.
  • Ma taką samą nazwę jak klasa.
  • Nie zwraca wartości.
  • Może przyjmować argumenty.

4. Konstruowanie obiektu

        var osoba = {
            nazwisko: "Kowalski",
            imie: "Jan",
            zawod: "piekarz",
            wyswietl: function () {
                document.write(this.nazwisko + " " + this.imie);
            }
        };

5. Znaczenie słowa this

Pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu.

6. Konstruktor w JavaScript

        function Klient(nazwisko, imie, zawod) {
            this.nazwisko = nazwisko;
            this.imie = imie;
            this.zawod = zawod;
            this.wypisz = function () {
                alert(this.nazwisko + ' ' + this.imie);
            };
        }

7. Znaczenie słowa kluczowego new

        var osoba1 = new Klient("Kowalski", "Jan", "kierowca");

8. Znaczenie słowa kluczowego prototype

        function Klient() {
            this.nazwisko = 'Bielski';
            this.imie = 'Paweł';
        }
        Klient.prototype.piszDane = function () {
            document.write(this.nazwisko + ' ' + this.imie);
        };

Prototype pozwala na dodawanie metod i właściwości do istniejących obiektów.
