Informacje o pętlach

Pętla for

Nazwa pętli: Pętla for

Konstrukcja:

        for (inicjalizacja; warunek; aktualizacja) {
            // kod do wykonania
        }

Opis:

Pętla for jest używana do wykonywania bloku kodu określoną liczbę razy. Inicjalizacja ustala początkową wartość zmiennej sterującej, warunek określa, kiedy pętla powinna się zakończyć, a aktualizacja zmienia wartość zmiennej sterującej po każdej iteracji.

Pętla while

Nazwa pętli: Pętla while

Konstrukcja:

        while (warunek) {
            // kod do wykonania
        }

Opis:

Pętla while jest używana do wykonywania bloku kodu, dopóki warunek jest spełniony. Warunek jest sprawdzany na początku każdej iteracji.

Pętla do-while

Nazwa pętli: Pętla do-while

Konstrukcja:

        do {
            // kod do wykonania
        } while (warunek);

Opis:

Pętla do-while jest podobna do pętli while, ale warunek jest sprawdzany na końcu każdej iteracji. Oznacza to, że blok kodu jest wykonywany przynajmniej raz, nawet jeśli warunek jest fałszywy od początku.

Poniżej znajdują się przykłady użycia tych pętli w praktyce:

Przykłady użycia pętli

        // Pętla for
        for (let i = 0; i < 5; i++) {
            console.log(i);
        }

        // Pętla while
        let j = 0;
        while (j < 5) {
            console.log(j);
            j++;
        }

        // Pętla do-while
        let k = 0;
        do {
            console.log(k);
            k++;
        } while (k < 5);
