Wczytanie końca pętli z użyciem instrukcji-okienka prompt

Przykład użycia instrukcji-okienka prompt do wczytania końca pętli:

        
        let koniec_petli = prompt("Wpisz tekst kończący pętlę:");
        for (let i_pierwsza_litera_nazwiska = 0; i_pierwsza_litera_nazwiska < 10; i_pierwsza_litera_nazwiska++) {
            if (i_pierwsza_litera_nazwiska == koniec_petli) {
                break;
            }
            console.log("Przykładowy tekst: " + i_pierwsza_litera_nazwiska);
        }

Zapewnienie wyświetlania polskich liter w standardzie UTF-8

W powyższym przykładzie użyto deklaracji <meta charset="UTF-8"> w sekcji <head>, aby zapewnić wyświetlanie polskich liter w standardzie UTF-8.
